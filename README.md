# Simple Tap Game

A simple browser-based tap game where players tap a blue circle to score points within a 10-second time limit.

## Game Rules

- Tap the blue circle to increase your score
- The first tap starts a 10-second countdown timer
- When time runs out, a "Game Over" modal displays your final score
- Click "Restart" to play again

## How to Play

1. Open `index.html` in your web browser
2. Click the blue circle to start playing
3. Try to get the highest score possible in 10 seconds!

## Requirements

- Modern web browser with JavaScript enabled
- Node.js 20 (for development environment)

## Features

- Responsive design with smooth animations
- Visual feedback on tap interactions
- Game Over modal with restart functionality
- Clean, modern UI with gradient background

## Technical Implementation

- Single HTML file with embedded CSS and JavaScript
- No external dependencies
- Under 200 lines of code
- Compatible with all modern browsers
