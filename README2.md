# Simple Tap Game - Two Players

A competitive two-player tap game where players compete to get the highest score within a 10-second time limit.

## Game Rules

- Two players face off: Player 1 (Blue Circle) vs Player 2 (Red Circle)
- Tap your circle to increase your score
- The first player to tap starts the 10-second countdown timer
- When time runs out, the player with the higher score wins
- If scores are equal, it's a draw
- Click "Play Again" to start a new round

## How to Play

1. Open `index2.html` in your web browser
2. Player 1 uses the blue circle (left side)
3. Player 2 uses the red circle (right side)
4. Either player can tap their circle to start the game
5. Compete to get the highest score in 10 seconds!

## Requirements

- Modern web browser with JavaScript enabled
- Node.js 20 (for development environment)
- Two players sitting at the same device

## Features

- Real-time dual-player scoring
- Visual distinction between players (Blue vs Red)
- Winner determination and draw detection
- Responsive design for various screen sizes
- Smooth animations and visual feedback

## Game Outcomes

- **Player 1 Wins**: Blue circle player has higher score
- **Player 2 Wins**: Red circle player has higher score
- **Draw**: Both players have the same score
